# 💀 HORROR — Find Your Fear

A dark, cinematic horror movie database with 80+ movies from 30+ countries and languages.

## 👤 Author
**Tanzim Hossain Ayon** | CSE Graduate · BRAC University · IEEE Researcher
🌐 [Portfolio](https://tanzimhossainayon.vercel.app)

## 🚀 Features
- 💀 80+ horror movies from 30+ countries
- 🔍 Real-time search by title, director, country, genre, tags
- 🎭 Filter by sub-genre, country, language, year, rating
- 📊 Grid & list view toggle
- 🎬 Movie detail modal with IMDb link
- 🩸 Blood drip animation on load
- ✨ Red particle background
- 👁️ Glitch text effect
- 🖱️ Custom blood-red cursor
- 💾 Built with vanilla JS — no frameworks

## 🌍 Countries Included
USA, Japan, South Korea, Italy, Spain, France, UK, Australia, Mexico, Sweden, Thailand, India, China, Indonesia, Argentina, Canada, Germany, Russia, Norway, Denmark, Ireland, Philippines, Hong Kong, Turkey, New Zealand, Finland, Czech Republic, Poland, Taiwan, Bangladesh, Nigeria & more!
